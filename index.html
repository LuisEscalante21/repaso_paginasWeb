¿Qué es HTML y cuál es su función principal en el desarrollo web?

Es un lenguaje marcado de hipertexto utilizado para el desarrollo de paginas web y su principal función seria darle estructura a estas mismas.

¿Qué son las etiquetas HTML y cómo se utilizan?

Son los que se utlizan para dar estructura u formato a una pagina web y se utilizan como instrucciones que indican como debe verse o comportarse el contenido.

Volver

Ejemplos de algunas de las etiquetas HTML más comunes.

  • h1, h2 hasta h6 para crear títulos de párrafos, de más importante h1 a menos h6.

  • p Los párrafos de texto de nuestro documento.

  • br Un salto de línea simple.

  • hr Salto de línea que indica un cambio de temática entre párrafos.

  • li para crear una lista de elementos.

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

Es un lenguaje que permite crear la parte visual de paginas web ya estructurado y escrito en un lenguaje de marcado que en este caso seria HTML utilizado para diseñar y dar estilo a las páginas web, por ejemplo, alterando la fuente, color, tamaño y espaciado del contenido.

Volver

¿Cuáles son las propiedades más importantes de CSS?

  • Font-family: Define la tipografia.

  • Font-size: Define el tamaño de la fuente y el valor se puede escribir en pixels o en ems.

  • Color: Define el color de la tipografía.

  • Width: Define el ancho de un elemento, el valor se puede escribir en pixels, ems o porcentaje.

  • Max-width o min-width: Definen el ancho máximo o mínimo de un elemento. Muy importante en sitios adaptables.

  • Height: Define el alto de un elemento, el valor se puede escribir en pixels, ems o porcentaje.

  • Max-height o min-height: Definen el alto máximo o mínimo de un elemento. Muy importante en sitios adaptables.

  • Padding: Es la distancia desde el borde de un elemento hasta su contenido.

  • Margin: Es la distancia entre un elemento y otro (desde el borde de un elemento hacia afuera).

  • Border: Define el borde de un elemento, su color, su estilo y grosor.

  • Background: Define los fondos de un objeto. El fondo puede ser una imagen o un color. El color puede ser pleno o degradado. La imagen se puede repetir formando una trama (es lo que ocurre por defecto) o se puede especificar que no repita y que se coloque en determinada posición.

Explicación del modelo de caja (box model) en CSS.

Cuando hablamos del modelo de cajas en CSS, es como un sistema que tiene el navegador de interpretar las diferentes partes de lo que solemos denominar «caja»: un elemento HTML con unas ciertas dimensiones.

¿Qué es JavaScript y cómo se integra en el desarrollo web?

Es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas y se integra en el head o en el body del HTML con la etiqueta script.

Volver

Explicación de cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).

El DOM es como un mapa estructurado de un documento HTML, si imaginamos que nuestra página web es un árbol gigante. Cada etiqueta HTML es una ramita en ese árbol. para interactuar con el HTML y el CSS, permitiéndote cambiar el contenido y el estilo de tu página web en tiempo real.

Volver

Algunos ejemplos de cómo JavaScript puede hacer que una página web sea más interactiva:

Validación de Formularios

Puedes usar JavaScript para validar formularios en tiempo real y asegurarte de que los datos ingresados sean correctos antes de enviarlos.

Mostrar/Ocultar Contenido

Puedes alternar la visibilidad de un elemento cuando se hace clic en un botón.

Cambiar Estilos Dinámicamente

Puedes cambiar el estilo de un elemento basado en interacciones del usuario, como hacer clic en un botón.

Realizar Cálculos

Puedes realizar cálculos en tiempo real basado en la entrada del usuario.

Animaciones

Puedes agregar animaciones simples usando JavaScript y CSS.

¿Qué es Git y por qué es importante para el desarrollo web?

Git es una herramienta esencial para los desarrolladores web porque les permite rastrear cambios en su código, colaborar con otros sin conflictos, trabajar en nuevas funcionalidades mediante ramas, integrar y probar automáticamente los cambios, y revisar el historial detallado de modificaciones, todo lo cual facilita la gestión eficiente del desarrollo de software.

Volver

Explicación de cómo Git se usa para el control de versiones en proyectos de desarrollo web.

Git permite a los desarrolladores web rastrear cambios en el código, colaborar sin conflictos y trabajar en nuevas funcionalidades mediante ramas, asegurando un desarrollo eficiente y ordenado del proyecto. También facilita la integración de cambios y la revisión del historial, garantizando que el proyecto se mantenga estable y bien gestionado.

Volver

¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

GitHub es una plataforma en la nube que permite a los desarrolladores colaborar en proyectos web. Utiliza Git para controlar versiones, permite trabajar en equipo en diferentes ramas, facilita la revisión de cambios a través de "pull requests", y ofrece herramientas para gestionar tareas y errores, mejorando así la calidad y eficiencia del desarrollo del software.
